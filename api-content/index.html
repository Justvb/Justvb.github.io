二三 奋力前行的我们逆流而上，却又不断地被浪潮推回到过去。车窗外闪过一片废墟，那里将会有一条路，那里曾是我居住十年的家。我们不免感怀，却又只好继续缅怀之际面向未来。 多么简单啊！一盘炸鸡腿，一份好成绩，一起玩耍的好朋友，一切都简单而美好，这或许是意义吧——对于一个孩子而言…… 多么简单啊，一个孩子拥有许多的欢乐，相应地当他成长后，一些烦恼会不会相伴而来呢？收回思绪，下了车。我穿过人来人往的广场，像个观众与过客。寻得一处台阶坐下，看那碌碌众人，都在追寻着什么吧。 一 宇宙恒常，一片片过往为什么就一定会是水波一样合拢在现实之后，而不能寄存于平行之宇宙呢？当我在夜晚爬下床饮水看到窗外的那一束光不能是若干年前日光灯映在我眼里的那一份印象呢？当我躺回床上时，床的另一侧不能是那个夏日时光浸泡着的午后躺在床上同样在思索的我呢。 所以，意义呢？ 不要紧，我来告诉你。 我知道自己在思索，我知道自己在思考万千事物渺茫之组成，既知一粒沙里见世界，一朵花里见天国，无限永远不能被有限所反映，时光也不会为我所停滞；那么意义，于我而言就是我在有限的认知范围内去见证去认知世界，然后收拾好我的所有同众人一同去追赶着各自的追求…… 二 人群中挤过来一个熟悉的身影，稚嫩的脸上挂着笑容，“走，一起去玩？”“嗯？”，我呆滞了一下，但看着他不能再熟悉的面孔，“走。”我说。
